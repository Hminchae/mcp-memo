#Ai 

> MCP 는 LLM에 Context 를 제공하는 방식을 표준화 하는 개방형 프로토콜로, Ai 모델을 다양한 데이터 소스 및 도구에 양방향 연결을 구축할 수 있는 표준화 된 방식을 제공함

- LLM 과 도구의 표준화된 연결 : AI 모델이 다양한 데이터 소수 및 도구와 원활하게 통합될 수 있도록 표준화된 프로토콜을 제공함
- 사전 구축된 다양한 컨텍스트 도구 연결 : 이미 구축되어 있는 MCP 서버들의 기능을 활용하여 LLM이 외부 시스템 및 컨텍스트에 빠르고 쉽게 연동할 수 있음
- 유연하고 확장 가능한 시스템 : 호스트 - 클라이언트 - 서버로 구성된 3계층 아키텍처를 통해 유연하고 확장 가능한 시스템을 구현할 수 있음
